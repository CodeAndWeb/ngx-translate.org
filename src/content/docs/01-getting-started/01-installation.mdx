Start by adding the npm module:

<Tabs syncKey="packetmanager">
    <TabItem label="npm">
```bash
npm install @ngx-translate/core --save
```
    </TabItem>
    <TabItem label="yarn">
```bash
yarn add @ngx-translate/core
```
    </TabItem>
</Tabs>
